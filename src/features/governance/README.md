# Feature: Governance (Governança Fiscal)

## Descrição
Sistema de governança fiscal e compliance com integração da Prefeitura de Campos Verdes como Oracle Fiscal.

## Componentes
- `OracleFiscalDashboard` - Dashboard do Oracle Fiscal (Prefeitura)
- `GovernancePage` - Página principal de governança
- `ComplianceDashboard` - Dashboard de compliance e métricas fiscais

## Responsabilidades
- Emissão de Notas Fiscais via Prefeitura (Oracle Fiscal)
- Rastreamento de arrecadação de CFEM e royalties
- Compliance com ANM (Agência Nacional de Mineração)
- Transparência fiscal e auditabilidade

## APIs e Integrações
- API Prefeitura de Campos Verdes (Oracle Fiscal)
- Sistema de Notas Fiscais Eletrônicas
- ANM (Agência Nacional de Mineração)
- EAS Protocol para atestações fiscais on-chain
